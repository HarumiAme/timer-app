# Temporizador con ReactJS
[Live Version](https://timer-js-d975c0acbe31.herokuapp.com)
### Descarga repositorio
```
git clone https://github.com/HarumiAme/timer-app.git
```
### Entra en directorio
```
cd timer-app
```
### Inicia la app
```
npm start
```
